🕐 Shift Management

Create as many shifts as you need — each with its own name, hours, time zone, and assigned employees. An employee can only be assigned to one shift at a time.

📊 Report Generator

Covers employees, loads, attendance, late %, away %, audit summary and top performers for the period you pick — then export as PDF or CSV.

🗂 Monthly Archive

Every completed month from the company-wide Monthly Statistics panel, kept permanently. The current month stays live until it ends and is archived automatically.


☁️ GitHub Auto-Backup (Developer Only)

Optional. When enabled, every newly-archived month is also pushed to a GitHub repo as a JSON file. ⚠️ The token below is stored in Firestore — use a fine-grained, repo-scoped token only, and lock down firestore.rules before relying on this.









⚠️ Violations Log

Every Approve (excused, no penalty) or Deny decision made on a Late Clock-in, Extended Away, or Extended Break alert. Dismissing an alert never creates an entry here.

🗂 Monthly Archive

Every completed month, kept permanently. Resets each month — the current month closes out automatically once it ends. A PDF + Excel export is also attempted automatically during the last 24 hours of every month, from whichever admin session happens to be open.


☁️ GitHub Fallback Backup (Developer Only)

Optional. Only used as a fallback — if no admin session was open to auto-download the night before month-end, the finalized month is pushed here instead as a JSON file. ⚠️ The token below is stored in Firestore — use a fine-grained, repo-scoped token only, and lock down firestore.rules before relying on this.
